# 五子棋技能系统下线功能说明书 (Functional Spec)

## 1. 业务逻辑描述
为了简化五子棋游戏体验，现决定移除五子棋的技能系统及相关的 UI 控制开关（灯泡）。

## 2. 修改范围
### 2.1 游戏大厅 (Lobby)
- 移除“技能五子棋”卡片上的技能开关灯泡（💡）。
- 移除大厅中与五子棋技能配置相关的同步逻辑。

### 2.2 游戏界面 (In-game)
- 隐藏/移除游戏界面下方的技能栏（包括“飞沙走石”、“力拔山兮”、“时光倒流”）。
- 确保技能能量轴不再显示（可选，视 UI 整洁度决定，目前计划随技能一并移除）。

## 3. API 与逻辑变更
- 默认将 `skillEnabled` 设置为 `false`。
- 停止向后端发送 `skillEnabled` 的变更请求。

## 4. UI 交互流程
- 用户点击五子棋卡片直接进入游戏，不再有灯泡交互，进入游戏后看到的是纯净的棋盘界面。

# 五子棋技能系统下线交互说明书 (Interaction Spec)

## 1. 交互变更
- **大厅界面**：
    - 旧：点击灯泡切换技能开关。
    - 新：灯泡消失，点击卡片仅表示选中游戏。
- **游戏界面**：
    - 旧：下方显示三个技能按钮及能量值。
    - 新：技能按钮及能量值面板消失。

## 2. 边缘案例处理
- **旧配置残留**：如果服务器缓存了 `skillEnabled: true` 的房间状态，前端渲染器应强制忽略该状态，确保不渲染技能 UI。
